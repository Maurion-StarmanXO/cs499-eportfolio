Algorithms and Data Structures

Artifact: CS-340 Project Two Animal Shelter Dashboard

This artifact comes from my CS-340 Project Two dashboard. The project used Python, MongoDB, Dash, and Plotly to create an interactive dashboard for animal shelter data. I selected this artifact for the algorithms and data structures category because it required filtering, organizing, processing, and presenting data in a way that users could interact with.

Enhancement Overview

For this enhancement, I improved the dashboard logic used to filter and display records. I worked on making the filtering behavior more reliable and easier to use, including support for rescue categories and reset behavior. I also improved how the dashboard handled data coming from MongoDB so that the user interface could update based on the selected criteria.

Skills Demonstrated

This artifact demonstrates algorithmic thinking through conditional filtering, query construction, data transformation, and dashboard callbacks. The project required me to think about how data moves from the database into the application, how filters change the dataset, and how the visual components respond to user actions.

Course Outcomes Supported

This enhancement supports the course outcome related to designing and evaluating computing solutions using algorithmic principles and computer science practices. It also supports professional communication because the dashboard needed to present information clearly and logically to users.
